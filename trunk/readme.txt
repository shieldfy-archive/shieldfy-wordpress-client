Shieldfy is a cloud-based security shield for your website to protect it from web attacks and malwares.

== Description ==


= Start Protecting Your Website Block attacks targeting your website. =

Shieldfy works as an external shields loads before your website and filter all requests , passing only trusted non harmful traffic and block other malicious traffic

Shieldfy Engine can identify and block several attacks including and not limited to
XSS(cross site scripting) , SQL Injection , Local/Remote File Inclution and many other


= Web Malware Detection & Cleaning IN-Code Scanner =

With our In Code Scanner , We scan your website for Web Malwares & Shells that harms your website

Our engine is smart enough to detect not only known malwares via traditional signature but also it detect unknown malwares by tracking suspicious code


= Ban Ip's Block any ip. =

You can specify single ip or multiple ips to block its access to your website and our firewall will take care of them.

Shieldfy Client can block any ip or list of ip's according to your needs

= Fast High level support =

Shieldfy security team is always here for help , our support is here for you anytime 24/7.


== Installation ==

Installation required to register to https://shieldfy.com to get your token

1. install your plugin and activate it
1. go to [Shieldfy.com](https://shieldfy.com/auth/register "Register now for free") and create a new account
1. add your website , you will see a popup choose wordpress and copy your token
4. in the plugin page paste your token and click Activate

== Frequently Asked Questions ==

= What is that plugin do ? =

shieldfy plugin protects you from web attacks like xss , sql injection , local file inclusion ... etc and scan your website to detect malwares and backdoors

= is this plugin for free =

Yes, shieldfy plugin is for free altough it connects you with shieldfy service which has a free plan and paid plan , you can choose what you need.

= Will Shieldfy Security slow my site down? =

No , Shieldfy uses caching on both your website and our endpoint server to improve the prefomance , we only take from 30 to 40 millisecond to analyze the requests.
